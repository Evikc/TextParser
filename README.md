# 📄 TextParser - OCR Scanner for Documents

**TextParser** — это современное мобильное приложение для Android, которое превращает ваш смартфон в мощный сканер документов. Оно использует технологию оптического распознавания символов (OCR) для извлечения текста из изображений, полученных с камеры устройства.

## ✨ Features

- **📸 Real-time Camera**: Встроенный просмотр камеры с возможностью захвата изображения одним касанием.
- **🔍 OCR Recognition**: Высокоточное распознавание печатного текста на более чем 50 языках с помощью Google ML Kit.
- **✍️ Text Editor**: Возможность редактировать распознанный текст прямо в приложении.
- **📋 Copy & Share**: Быстрое копирование текста в буфер обмена или отправка в другие приложения.
- **⚡ Offline First**: Все операции выполняются локально на вашем устройстве — никаких задержек и полная конфиденциальность.
- **🌙 Material You**: Адаптивный интерфейс, поддерживающий светлую и темную тему.

## 🚀 Getting Started

### 📋 Prerequisites

Перед запуском убедитесь, что у вас установлено следующее ПО:

- **Android Studio** (Hedgehog или новее)
- **JDK 11** или выше
- **Android SDK** с API Level 24 (Android 7.0) или выше
- Физическое устройство или эмулятор с камерой
